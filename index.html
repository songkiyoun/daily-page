v0.6.4

캐릭터 세팅

무기와 성격은 새 캐릭터 생성 시에만 선택합니다. 한 번 탑 등반을 시작하면 해당 런이 끝날 때까지 바꿀 수 없습니다.

스탯 설정

상점 예정

상점, 시작 강화, 최초 캐릭터 성장 기능은 전투 코어와 보상 구조가 안정된 뒤 추가합니다.

전투 시뮬레이터

현재 플레이어 세팅과 선택한 상대 조합을 1층 기본 스탯 5 기준으로 빠르게 반복 테스트합니다.

아직 시뮬레이션을 실행하지 않았습니다.

개발 원칙

  • 새 패치 블록을 아래에 덧붙이지 않습니다.
  • 수정은 담당 파일의 기존 함수와 데이터를 직접 고칩니다.
  • 1:1 전투만 고려합니다.
  • 맵은 사각형 전장입니다.
