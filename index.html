처음 사용하시면 Google Apps Script를 연결해주세요. 한 번만 설정하면 PC·모바일 데이터가 자동으로 동기화됩니다.

1
script.google.com → 새 프로젝트 → 받은 AppScript.gs 코드 붙여넣기
2
배포 → 새 배포 → 유형: 웹 앱 → 액세스: 모든 사용자 → 배포
3
생성된 웹 앱 URL을 아래 입력
Apps Script 웹 앱 URL
올바른 Apps Script URL을 입력해주세요.
🔒 URL은 이 브라우저에만 저장됩니다. 같은 URL을 다른 기기에도 입력하면 데이터가 공유됩니다. URL 변경은 상단 🔗 버튼을 누르세요.
